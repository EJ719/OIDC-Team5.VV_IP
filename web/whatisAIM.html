What is AIM?





AIM [ Auto Internet protocol address Manager ]

AIM은 IP Address를 수작업으로 관리할 때 생기는 이슈들을 해결하기 위해 런칭되었습니다.

실시간 모니터링과 로깅을 통해 보안 유출을 줄이며 잠정적 재정 타격으로부터 보호할 수 있습니다.

자동화된 간편 시스템으로 기업 뿐 아니라 개인도 쉽고 자유롭게 활용 가능합니다.

AIM의 IP Managing이 필요한 기업 또는 개인이 IP를 관리하기 관리자 계정을 요청을 하게 되면, 요청한 사람에 한해서 AIM은 관리자 계정을 배포합니다.

관리자 계정을 받게 되면 관리자 계정에서 사용자의 계정을 만들 수 있습니다. 사용자 계정을 만들게 되면, 이 계정이 사용하는 Network List를 불러올 수 있습니다.

또한 IP Address 중에서 아직 할당 받지 못한 IP Address의 List와 사용자 계정에서 할당 받지 못한 IP Address를 불러올 수 있습니다.
